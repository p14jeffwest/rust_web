* 프로젝트 전체 구상
  - 최종 목표: 순수 Rust 코드로 동작하는 웹 사이트 제작 및 서비스
  - 구성
    . 웹 서버 인프라: AWS의 Lightsail 인스턴스 (1G RAM, 30GB SSD) -> 가장 작은 인프라로
	. 웹 서버 프로그램: Rust의 Axum 패키지 이용해서 순수 코딩
	. 제공 서비스: 한자를 한글로 변환 (대상 소스: 문자열, 파일, 웹 페이지)
  - 필요 사항
    . Kwnology: Rust 코딩 능력
    . 필요 SW 도구: 개발용 PC, Rust 개발용 SW (Cargo, VS Code), git
    . 웹 서버 인프라용 비용: 도메인(3천원/1년), AWS Lightsail($5/월)
  - 이 프로젝트를 끝내면 습득되는 지식
    . Rust 기본 문법에 대한 탄탄한 이해
    . Rust를 이용한 웹서버 프로그래밍 지식
	. Rust를 이용한 비동기 프로그래밍 방법(tokio 이용 방법)
	. Rust를 이용한 웹 페이지 스크래핑 방법(scraper 패키지 이용 방법)
    . 한자 -> 한글 변환에 대한 방식 이해 
    . 기본적인 웹 프론트엔드 프로그래밍(HTML, CSS, JS)	
	. git 이용한 코드 관리
	. AWS Lightsail에서 웹 서비스 제공 방법(DNS관리, Service 생성/시작)
